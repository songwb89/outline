# 智能授课

## 页面结构

左侧教材目录，右侧内容区显示课件和课堂互动。

## 教材目录

- 顶部显示当前教材信息（名称、年级）
- 按章节展示目录树，支持展开/折叠
- 同时只展开一个章节
- 点击节次加载对应内容

## 教学课件

- 列表展示当前节次的课件
- 按**创建时间降序**排列
- 点击「上课」按钮弹出班级选择

### 上课流程

1. 点击「上课」弹出班级选择弹窗
2. 班级列表筛选：老师负责的班级中，**学科与当前教材所属学科一致**的班级
3. 选择班级后点击「进入上课」，打开授课页面
4. 可选「跳过，稍后选择」直接进入

## 课堂互动

- 按**创建时间降序**排列

### 评测记录

- 单词评测、对话评测、场景对话支持查看评测记录
- 评测记录按**评测时间降序**排列
- 点击记录卡片可查看评测结果
