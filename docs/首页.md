# 首页

## 页面布局

采用网格布局展示6个业务板块。

## 板块说明

### 1. 待办事项

与当前线上系统保持一致，无调整。

### 2. 最新诊断

与当前线上系统保持一致，无调整。

### 3. 模块掌握度

呈现雷达图，数据来源与数据分析界面一致。

数据的时间范围为全部数据。

**选择器**：

- 班级/学科采用扁平化选择器，格式为「班级（学科）」
- 默认选中任意一个，无规则要求
- 用户切换后，缓存至浏览器本地，下次进入，默认为用户最后选中的。

### 4. 得分率趋势分析

呈现折线图，数据来源与数据分析界面一致。

数据的时间范围为全部数据。

**选择器**：

- 班级/学科采用扁平化选择器，格式为「班级（学科）」
- 默认选中任意一个，无规则要求
- 用户切换后，缓存至浏览器本地，下次进入，默认为用户最后选中的。

### 5. 学业预警

显示老师所带所有班级学生中，得分率最低的5个（跨班级排名）。

**筛选规则**：

- 最近5次作业平均得分率低于70%
- 取得分率最低的5个学生
- 按得分率升序排列

**显示内容**：

- 学生头像
- 学生姓名
- 班级（学科）
- 最近5次作业平均得分率

### 6. 课堂评价排行榜

显示老师对学生的评价标记排名。

**数据范围**：老师所教的所有班级中的学生（跨学科）

**说明**：若老师同时教授某班级语文和数学，同一学生在两个学科的标记数据累计，不做去重。

**榜单内容**：

- 点赞TOP5：学生姓名、班级、点赞次数
- 待改进TOP5：学生姓名、班级、待改进次数

**排名规则**：按对应标记的累计次数降序排列
