# 运营端 - 文件管理

## 筛选条件

支持按以下条件筛选资源列表：

| 条件     | 类型 | 说明                                |
| :------- | :--- | :---------------------------------- |
| 关键词   | 文本 | 匹配资源名称、创建人                |
| 资源类型 | 下拉 | 课件 / 教案 / 学案                  |
| 文件格式 | 下拉 | doc / docx / pdf / ppt / pptx / txt |
| 状态     | 下拉 | 正常 / 检测中 / 违规 / 待审核       |
| 学校     | 文本 | 匹配学校名称                        |

支持重置和查询操作。

## 资源列表

### 列表字段

序号、资源名称、资源类型、文件格式、状态、创建人、学校、创建时间

### 状态说明

| 状态   | 标签颜色 |
| :----- | :------- |
| 正常   | 绿色     |
| 检测中 | 蓝色     |
| 违规   | 红色     |
| 待审核 | 黄色     |

## 操作

| 操作 | 说明                                                     |
| :--- | :------------------------------------------------------- |
| 查看 | 打开预览弹窗                                             |
| 删除 | 二次确认后删除                                           |
| 审核 | 仅"待审核"状态显示，点击打开审核弹窗，可选择通过或不通过 |

## 审核操作

- **通过**：状态变为"正常"
- **不通过**：状态变为"违规"
